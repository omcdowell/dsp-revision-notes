# DSP Revision Notes

Live revision notes for CS6004 (Digital Signal Processing), rendered in the browser with LaTeX math support. Zero build step — static site served via GitHub Pages.

- Notes live in `notes/*.md` (Markdown + LaTeX).
- `notes/manifest.json` controls the sidebar.
- Open `index.html` via any static server (e.g. `python3 -m http.server`) to run locally.
